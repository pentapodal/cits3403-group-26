## Contributing
### Setup
1. Clone the repository
2. Launch a terminal within the directory
3. Create a virtual environment with `python3 -m venv venv`

### Development
4. Activate the virtual environment with `source venv/bin/activate` (Linux) or `.\venv\Scripts\Activate.ps1` (Windows)
5. Install the packages in requirements.txt with `pip install -r requirements.txt`
6. Create a new branch for your patch
7. Upgrade the database with `flask db upgrade`
8. Make whatever edits you want
9. Update requirements.txt with `pip freeze > requirements.txt`
10. Commit and push the new branch to GitHub
11. Create a pull request for the new branch


## Overshare Introduction 
Overshare is a web application that lets users upload their Instagram data and generate a recap of their year on Instagram. By analyzing your interactions—likes, comments, and stories—you get a clear overview of your most active moments and connections. You can also explore friends' recaps and compare insights in a fun, interactive way.
Whether you're just curious or ready to flex your stats, Overshare makes it easy to reflect, recap, and, well... overshare.
